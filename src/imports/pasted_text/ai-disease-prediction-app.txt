You are a senior full-stack AI engineer. Build a complete MVP production-ready application for an AI Early Disease Prediction System.

The system predicts diseases like Diabetes and Heart Disease using trained machine learning models.

The ML models (Random Forest, SVM, Naive Bayes, Decision Tree) are already trained and saved as .pkl files.

The backend must load the best model and scaler to perform predictions.

BACKEND REQUIREMENTS

Use FastAPI.

Create endpoint:

POST /predict

Input schema:

{
pregnancies: number
glucose: number
blood_pressure: number
skin_thickness: number
insulin: number
BMI: number
diabetes_pedigree: number
age: number
}

Load model:

model = joblib.load("ml/models/best_model.pkl")
scaler = joblib.load("ml/models/scaler.pkl")

Process:

Convert input to numpy array

Scale using scaler

Predict using model

Return probability and prediction

Return response:

{
prediction: 0 or 1
risk_probability: float
}

Add CORS middleware.
